Paso a paso

Desde 0

  1. git init en la terminal
  2. git add .
  3. git commit -m "nombre del commit"
  4. git branch -M main
  5. git remote add origin https://github.com/MauricioPerezCaram/xxx
  6. git push -u origin main

Debería estar subido todo

Para View Deployment

Settings > Pages > Branch > Main > Save

Para meter un cambio a algo ya empezado

  1. Hago todos los cambios
  2. Guardo con Ctrl S
  3. Guardo con git add .
  4. git commit -m "nombre del cambio"
  5. git branch -M main
  6. git push

Y si quiero descargar uno para modificarlo

  1. Entro a Github
  2. Perfil, repositorio, code (copiamos link)
  3. Abro carpeta donde quiero descargar el archivo
  4. Click derecho git bash
  5. Git Bash > git clone (link que copiamos)
  6. Tenemos archivos para romper y despues hacemos los pasos de mas arriba para subir cambios a github
